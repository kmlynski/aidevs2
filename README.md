# AI Devs 2

Repo that uses Playwright test runner with TypeScript and axios for the second edition of [AI Devs](https://www.aidevs.pl/).

## Getting started

# Install dependencies

```bash
npm i
```

# Add .env file with:

- AI_DEVS_API_KEY
- AI_DEVS_API_BASE_URL

# Run tests

```bun
npm run <script_name>
```

Example:

```npm
npm run C02L02
```
